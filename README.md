This repository contains my work for the Dacon 2025 Power Consumption Prediction AI Competition (2025 전력사용량 예측 AI 경진대회).
The goal of this competition was to predict future electricity usage based on time-series data collected from smart factory equipment.

Through this project, I explored time series forecasting techniques, model building, and feature engineering strategies tailored for energy consumption data.

I completed this project by combining my own ideas, external resources, and hands-on experimentation. This approach allowed me not only to implement working models but also to deeply understand the reasoning behind each step.
